числовые типы
uint,int,float, double
Тип int используйте для натуральных чисел, а double — для веществен-
ных. Тип decimal применяйте для денежных расчетов, измерений в чер-
тежах и машиностроительных схемах и повсюду, где важна точность
вещественных чисел.
